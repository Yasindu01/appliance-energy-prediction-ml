# Appliance Energy Prediction — ML Final Project

**Course:** Machine Learning / Artificial Intelligence  
**University:** University of Messina  
**Student:** Kahathota Liyanage Yasindu Janiya Weerasinghe  

## Overview
This project develops a supervised machine learning pipeline 
to predict household appliance energy consumption using 
environmental and temporal data from a low-energy house 
in Belgium (Candanedo et al., 2017).

## Dataset
UCI Appliances Energy Prediction Dataset  
19,735 observations | 29 features | 10-minute intervals

## Models Compared
- Linear Regression (baseline)
- Decision Tree Regressor
- Support Vector Regression (SVR)
- Random Forest Regressor (best model)

## Best Result
Random Forest — Test R²: 0.5305 | MAE: 33.07 Wh
